Recommendations

The recommendations for the users are their vector’s distance from each item vector:

 

·         User1 would be most interested in Item1, Item3, and finally Item2.

·         User2 has interests the same order, but would find the whole list less interesting.

·         User3 would be most interested in Item2, then Item3, then Item1.

 

Item-Item Similarity

The Item-Item similarity set is the distances between the item vectors.

·         Item2:Item3 = 0.15, Item1:Item3 = 0.2, and finally Item1:Item2 = 0.35

 

User Vectors

The User-User distances are random, there is nothing to be learned.

 

Summary

The Semantic Vectors algorithm takes a matrix of Row->Column relationships and creates a set Row-Column distances and a set of Column-Column relationships in a new, common numerical space.  In this example, we created two sets of recommenders, a User->Item recommender and an Item->Item recommender.
